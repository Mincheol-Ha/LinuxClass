# 환경설정
>> 안드로이드 스튜디오 상에서 프로젝트 닫을때는 File > Close Project 이용
>> 마우스 + 휠로 글자 크기 변경
>> 한글 글꼴 변경 (settings > editor > font > fallback)

>> 가입시 작성하는 회원 정보는 아이디/비밀번호/이름/이메일
>> 자동으로 작성되는 정보는 회원번호/가입일
>> 데이터베이스는 sqlite를 사용
